# Node HTTP Server From Scratch

This repository contains multiple backend projects built using **pure Node.js**
without Express or any frameworks.

## Projects
1. Basic HTTP Server:
Basic HTTP Server
Built using Node.js native http module
Handles GET & POST requests
Manual routing and JSON responses

2. File Based CRUD Server 
This is a simple CRUD (Create, Read, Update, Delete) API built using only Node.js (no Express or other frameworks). It stores user data in a JSON file and handles HTTP requests to perform operations on that data. The code is organized into multiple files and folders to make it easier to read and maintain.

# Features
Get all users

Get a single user by ID

Add a new user

Update an existing user

Delete a user

Data is saved in Data.json and persists between server restarts.





## Installation
Clone or download this project folder to your computer.

Open a terminal in the root folder (where package.json is located).

Install dependencies (there are none for the core server, but you might want to initialize npm):

bash
npm init -y
This will create a package.json file (you can skip if you already have one).

The project is ready to run.
